# ESP32-S3 轮腿机器人引脚分配说明

## 文档说明

本文档用于记录当前主控板的引脚分配结果，依据为原理图以及后续人工确认的修正信息。

- 主控芯片：`ESP32-S3`
- 用途：作为后续固件引脚宏定义、外设初始化和驱动开发的统一依据
- 说明：本文档中的信号命名保持与原理图一致，便于硬件和软件对照

## 1. 主控与系统模块

这一部分包含启动、复位、USB 下载等基础系统信号。

| 信号名 | ESP32-S3 引脚 | 说明 |
| --- | --- | --- |
| `BOOT` | `GPIO0` | 启动下载相关按键，属于启动配置脚 |
| `RST` | `EN` 引脚 | 芯片复位输入，不是普通 GPIO |
| `D-` | `GPIO8` | 原生 USB 下载接口负端 |
| `D+` | `GPIO19` | 原生 USB 下载接口正端 |

## 2. 电驱模块

这一部分统一整理电驱相关信号，包括电驱使能、故障反馈、三相 PWM 输出以及电驱反馈采样输入。

| 信号名 | ESP32-S3 引脚 | 说明 |
| --- | --- | --- |
| `EN` | `GPIO14` | 电驱使能信号，由主控直接控制 |
| `FAULT` | `GPIO21` | 电驱故障反馈输入 |
| `PWM_A_L` | `GPIO39` | 左轮电机 A 相 PWM 输出 |
| `PWM_B_L` | `GPIO40` | 左轮电机 B 相 PWM 输出 |
| `PWM_C_L` | `GPIO41` | 左轮电机 C 相 PWM 输出 |
| `PWM_A_R` | `GPIO48` | 右轮电机 A 相 PWM 输出 |
| `PWM_B_R` | `GPIO47` | 右轮电机 B 相 PWM 输出 |
| `PWM_C_R` | `GPIO42` | 右轮电机 C 相 PWM 输出 |
| `SQA_L` | `GPIO1` | 左侧电驱采样反馈输入，原理图网络名为 `SQA_L` |
| `SQB_L` | `GPIO2` | 左侧电驱采样反馈输入，原理图网络名为 `SQB_L` |
| `SQC_L` | `GPIO3` | 左侧电驱采样反馈输入，原理图网络名为 `SQC_L` |
| `SQA_R` | `GPIO4` | 右侧电驱采样反馈输入，原理图网络名为 `SQA_R` |
| `SQB_R` | `GPIO5` | 右侧电驱采样反馈输入，原理图网络名为 `SQB_R` |
| `SQC_R` | `GPIO6` | 右侧电驱采样反馈输入，原理图网络名为 `SQC_R` |

## 3. 编码器模块

当前编码器芯片为 `MT6835`，左右编码器共用一组 SPI 总线，分别使用独立片选。

| 信号名 | ESP32-S3 引脚 | 说明 |
| --- | --- | --- |
| `ENC_SPI_CLK` | `GPIO35` | 编码器 SPI 时钟 |
| `ENC_SPI_MISO` | `GPIO36` | 编码器 SPI 主输入从输出 |
| `ENC_SPI_MOSI` | `GPIO37` | 编码器 SPI 主输出从输入 |
| `CS_L` | `GPIO7` | 左编码器片选信号 |
| `CS_R` | `GPIO15` | 右编码器片选信号 |

## 4. IMU 模块

当前 IMU 为 `BMI088`，包含加速度计与陀螺仪两部分，采用共用 SPI 总线、独立片选方式连接。

| 信号名 | ESP32-S3 引脚 | 说明 |
| --- | --- | --- |
| `IMU_INT1` | `GPIO9` | IMU 中断输出 1 |
| `IMU_INT3` | `GPIO10` | IMU 中断输出 3 |
| `SPI_MOSI` | `GPIO11` | IMU SPI 主输出从输入 |
| `SPI_CLK` | `GPIO12` | IMU SPI 时钟 |
| `SPI_MISO` | `GPIO13` | IMU SPI 主输入从输出 |
| `CS_ACCEL` | `GPIO16` | BMI088 加速度计片选 |
| `CS_GYRO` | `GPIO17` | BMI088 陀螺仪片选 |

## 5. 舵机模块

当前舵机接口为单总线控制，仅保留一根数据线。

| 信号名 | ESP32-S3 引脚 | 说明 |
| --- | --- | --- |
| `DATA` | `GPIO18` | 舵机控制数据线 |

## 7. 使用备注

- 本文档已根据你后续确认的接线关系完成修正。
- 后续代码中的引脚宏定义建议严格以本文档为准，避免软件与原理图不一致。
- `EN` 在本文档中明确表示电驱使能信号，不是芯片复位 `EN` 引脚。
- 原理图中 `SQA/SQB/SQC` 这一组信号在软件文档中统一视为电驱采样反馈信号，并归类到电驱模块中管理。
- 编码器型号已确认为 `MT6835`。
- 左右电机三相 PWM 命名当前完全沿用原理图标注，后续若电机转向不符，应优先在控制层调整相序或方向定义，而不是直接改硬件命名。
